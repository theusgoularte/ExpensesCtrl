# ExpensesControl

ExpensesControl é uma aplicação web simples para controle financeiro pessoal. Ela permite adicionar receitas e despesas, visualizar o saldo atual e filtrar transações por mês e categoria. O site possui suporte a **português (pt-BR)** e **inglês (en-US)**, com placeholders, labels e botões traduzíveis.

---

## 💻 Tecnologias

- HTML5
- CSS3 (dark mode)
- JavaScript (vanilla)
- LocalStorage para persistência de dados
- Responsivo para diferentes tamanhos de tela

---

## 📝 Funcionalidades

1. **Adicionar transações**  
   - Tipo: Receita ou Despesa  
   - Descrição, Valor, Data, Categoria  

2. **Exibir resumo financeiro**  
   - Total de Receitas  
   - Total de Despesas  
   - Saldo (positivo/negativo com cores diferenciadas)  

3. **Filtrar transações**  
   - Por mês  
   - Por categoria  

4. **Apagar dados**  
   - Botão para limpar todas as transações  

5. **Suporte a idiomas**  
   - Botão de alternância **PT ↔ EN**  
   - Todos os textos e placeholders atualizam automaticamente

6. **Interface escura e responsiva**  
   - Layout compacto e moderno  
   - Colunas ajustáveis em telas menores

## 🚀 Como Usar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/ExpensesControl.git
```

2. Abra o index.html no navegador.
3. Adicione transações, filtre por mês/categoria ou altere o idioma usando o botão EN/PT.
4. Os dados ficam salvos automaticamente no navegador usando LocalStorage.
